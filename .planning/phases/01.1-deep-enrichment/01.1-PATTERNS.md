**Ollama call pattern** (qualify.py lines 31-50):
```python
def call_ollama(host_name: str, podcast_name: str, domain: str) -> QualifyResponse | None:
    user_msg = json.dumps({
        "host_name": host_name,
        "podcast_name": podcast_name,
        "domain": domain,
    })
    try:
        response = ollama.chat(
            model=CONFIG["model"],
            messages=[
                {"role": "system", "content": CONFIG["system_prompt"]},
                {"role": "user", "content": user_msg},
            ],
            format="json",
            options={"temperature": CONFIG["temperature"]},
        )
        parsed = json.loads(response["message"]["content"])
        return QualifyResponse(**parsed)
    except Exception:
        return None
```

Key details: `format="json"` is mandatory (prevents markdown fences), `options={"temperature": ...}` not `temperature=` at top level, `except Exception: return None` for graceful failure.
