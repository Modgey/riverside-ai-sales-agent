CONFIG_PATH = os.path.join(os.path.dirname(__file__), "prompts", "deep_enrich.json")

**Pydantic response model** (per D-09, D-12):
```python
class DeepEnrichResponse(BaseModel):
    podcast_themes: str
    company_summary: str
```

**Guest name extraction** (per D-03): define `GUEST_PATTERNS` list with these regex patterns:
- `r'\bwith\s+([A-Z][a-z]+(?:\s+[A-Z][a-z]+){1,2})'`
- `r'\bft\.?\s+([A-Z][a-z]+(?:\s+[A-Z][a-z]+){1,2})'`
- `r'\bfeat(?:uring)?\s+([A-Z][a-z]+(?:\s+[A-Z][a-z]+){1,2})'`
- `r'\|\s*([A-Z][a-z]+(?:\s+[A-Z][a-z]+){1,2})'`

Function `extract_guest_name(entry: dict) -> str | None`: iterate GUEST_PATTERNS against `entry.get("title", "")`, return first match group(1) or None. Do NOT use itunes_author at entry level (verified unreliable per RESEARCH.md Pitfall 1).

**Description cleaning**: Function `clean_description(raw: str, max_chars: int = 500) -> str`: strip HTML with `re.sub(r'<[^>]+>', ' ', raw)`, collapse whitespace, truncate to max_chars. Discretion: 500 chars balances richness vs data size.

**Episode details extraction** (per D-01, D-02): Function `extract_episode_details(rss_url: str) -> list[dict]`:
- Call `feedparser.parse(rss_url)`, iterate `feed.entries[:3]`
- For each entry, build dict with keys: `title`, `description` (cleaned via clean_description), `guest_name` (via extract_guest_name), `published` (from entry.get("published", ""))
- Return list of 3 dicts (or fewer if feed has fewer entries)
- Also extract up to 10 episode titles for theme analysis: `[e.get("title","") for e in feed.entries[:10]]`
- Return both as a tuple: `(episode_details_list, episode_titles_list)`

**Company scraping** (per D-04, D-05, D-06): Function `scrape_company_text(domain: str, timeout: int = 8) -> str`:
- GET `https://{domain}` with `headers={"User-Agent": "Mozilla/5.0"}` and timeout
- Parse with `BeautifulSoup(resp.content, "lxml")`
- Priority order: `meta[property=og:description]` content, `meta[name=description]` content, h1 text
- If homepage meta is empty, probe `https://{domain}/about` with same logic (per D-05, Claude's discretion)
- On any exception: return empty string (per D-06, no retries, prospect stays call-ready)

**Ollama combined call** (per D-07, D-08, D-09): Function `call_ollama_deep(episode_titles: list[str], episode_descriptions: list[str], raw_company_text: str) -> DeepEnrichResponse | None`:
- Build user_msg as JSON with keys: `episode_titles` (list, up to 10), `episode_descriptions` (list, up to 3 cleaned descriptions), `company_raw` (truncated to 800 chars)
- Call `ollama.chat(model=CONFIG["model"], messages=[system + user], format="json", options={"temperature": CONFIG["temperature"]})`
- Parse response with `json.loads`, validate with `DeepEnrichResponse(**parsed)`
- On any exception: return None (same pattern as qualify.py)

**Richer cheat sheet** (per D-11): Function `assemble_deep_cheat_sheet(prospect: ProspectDict) -> str`:
- Build dict with keys: `prospect` (host_name, company_name, company_size, industry, work_email, title), `podcast` (podcast_name, cadence_days, episode_count, podcast_themes), `recent_episodes` (from prospect["episode_details"]), `company_summary`, `scoring` (score, status)
- Return `json.dumps(data, indent=2)`

**Main entry function** (per D-10): Function `deep_enrich_prospects(prospects: list[ProspectDict]) -> list[ProspectDict]`:
- Split: `call_ready = [p for p in prospects if p.get("status") == "call-ready"]`, `others` = the rest
- Print: `f"  Deep enriching {len(call_ready)} call-ready prospects ({len(others)} others skipped)"`
- Track `scraped_fail = 0`, `llm_fail = 0`
- Loop over call_ready with per-item progress: `print(f"  Deep enriching {i+1}/{len(call_ready)}: {prospect.get('podcast_name', '')[:50]}")`
- For each: (1) extract_episode_details -> set prospect["episode_details"] and get episode_titles, (2) scrape_company_text -> get raw_company, increment scraped_fail if empty, (3) call_ollama_deep with episode_titles, episode descriptions from details, and raw_company, (4) if result: set prospect["podcast_themes"] and prospect["company_summary"], else: llm_fail += 1, set podcast_themes=None, company_summary = raw_company[:300] if raw_company else None, (5) rebuild cheat_sheet via assemble_deep_cheat_sheet
- Print summary: `f"  Deep enrich complete: {len(call_ready)} processed, {scraped_fail} scrape failures, {llm_fail} LLM failures"`
- Return `call_ready + others`

**`__main__` block**: minimal test that loads score.json (if exists) and runs deep_enrich_prospects on first 2 call-ready prospects, printing results.

| T-1.1-03 | I (Info Disclosure) | User-Agent header | accept | Generic Mozilla UA; no identifying info sent |
